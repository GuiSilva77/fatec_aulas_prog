SIGA

O Sistema Integrado de Gestão Acadêmica é um produto do Centro Paula Souza, desenvolvido para oferecer recursos administrativos e de gestão para suas Faculdades de Tecnologia e Escolas Técnicas Estaduais.

DIMENSÃO

O sistema é utilizado por 100 Unidades de Ensino e um público de 105 mil alunos. O sistema possui uma taxa de acesso diária de 1,5 milhão de solicitações.

ARQUITETURA

O sistema é desenvolvido em módulos, cada qual atendendo a um público específico de usuários e funcionalidades. O sistema é corporativo com sua base de dados integrada, tendo sido projetada para permitir a redução do retrabalho, e os processos são simples de serem executados.
